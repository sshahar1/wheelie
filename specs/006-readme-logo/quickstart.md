# Quickstart: README Logo

## Verify the change

1. Open `README.md` in this repo.
2. Confirm an image reference to `wheelie_logo.png` appears near the top, above or
   directly below the `# Dance Troupe Performance Bot` title.
3. Render the README to check the image displays correctly and at a bounded size:
   - On GitHub: push the branch and view the file, or open the PR diff preview.
   - Locally: open `README.md` in any Markdown preview (e.g., editor preview pane,
     `grip`, or VS Code's built-in Markdown preview).
4. Confirm no broken-image icon appears and the rest of the README content is
   unchanged below the inserted line.
